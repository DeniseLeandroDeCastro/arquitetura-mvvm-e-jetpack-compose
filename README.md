O app Desembaralhar, é um jogo de palavras embaralhadas e tem apenas um jogador. Uma palavra embaralhada é mostrada e o jogador precisa adivinhar, com base nas letras exibidas. 
O jogador marca ponto se a palavra estiver correta, pode tentar adivinhar quantas vezes quiser e também pode pular a palavra atual.
Este aplicativo foi criado com base no codelab (ViewModel e estado no Compose) da documentação oficial do Android e, claro, modifiquei o layout bem básico e adicionei uma carinha mais interessante.
